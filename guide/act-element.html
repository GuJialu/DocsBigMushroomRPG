# 剧情元素

# 激活条件

激活一个条件

参数 描述
条件名 条件的名字，制作者自定

# 移除条件

移除一个条件

参数 描述
条件名 需要移除的条件的名字

# 选项

给玩家选择的选项

  • 点击编辑
  • 点击“+”按钮添加选项
  • 输入选项内容
  • 输入该选项被选后将激活的条件

# 移动人物

移动某个人物到某个地点

参数 描述
人物名 移动人物的名字
地点名 目的地地点名

TIP

  • 如果移动的是主角，游戏会自动加载地图，移动游戏内主角人物
  • 随从人物会跟随主人移动

# 人物跟随人物

某个人物成为某个人物的随从

参数 描述
随从名 随从的名字
主人名 主人的名字

# 人物停止跟随人物

某个人物停止成为某个人物的随从

参数 描述
随从名 随从的名字
主人名 主人的名字

TIP

  • 停止跟随后，原随从人物会留在停止跟随时主人所在的地点

# 改变人物立绘

某个人物的立绘

参数 描述
人物名 人物的名字
图片文件名 新立绘图片文件名含后缀

# 对话

  • 点击编辑
  • 点击“+”按钮添加对话
  • 输入说话人物名字
  • 输入说话内容
  • 输入配音音频文件名（可选）
  • 输入特殊立绘图片名（可选）

# 展示图片

参数 描述
图片文件名 展示的图片文件名（含后缀），图片文件需在素材路径下
展示时长 展示的时长（秒）不填或0则为鼠标点击跳过

# 展示背景

展示一张图片，只能被移除背景移除。主要用于在对话/选项中添加背景。

参数 描述
图片文件名 展示的图片文件名（含后缀），图片文件需在素材路径下

# 移除背景

移除当前展示的背景。

# 更换BGM

参数 描述
音频文件名 音频文件名，不填则为停止播放当前BGM

# 交易

参数 描述
交易对象人物名

TIP

交易中购买物品需支付等于物品价值的金钱，出售则获得物品价值80%的金钱

# 获得金钱

某个人物获得金钱

参数 描述
人物名
金钱数

# 获得物品

某个人物获得物品

参数 描述
人物名
物品名

# 失去物品

某个人物失去物品

参数 描述
人物名
物品名

# 获得并装备物品

某个人物获得并装备物品，如果装备槽位已有装备，则旧装备放回背包中

参数 描述
人物名
物品名
装备槽位 包括衣，帽，武器一，武器二

# 停止装备物品

某个人物停止装备物品，并放回背包中

参数 描述
人物名
装备槽位 包括衣，帽，武器一，武器二

# 战斗

展开战斗

参数 描述
敌方首领人物名
死斗 战斗是否为死斗
胜利激活条件 战斗胜利将激活的条件
失败激活条件 战斗失败将激活的条件
背景图 战斗背景图片名含后缀

# 脱离当前战斗

脱离当前进行中的战斗，定位到某一场具体战斗见战斗中剧情

# 获得技能

参数 描述
人物名
技能名

# 失去技能

参数 描述
人物名
技能名

# 援兵

给当前进行中的战斗添加援兵，定位到某一场具体战斗见战斗中剧情

参数 描述
援兵阵营 我方/敌方
援兵人物名

# 切换主角

将某个人物切换为主角

参数 描述
人物名 切换为主角的人物的名字

TIP

切换主角后注意保证主角相关剧目的正常触发，见主角代号

# 杀死人物

杀死某个人物

参数 描述
人物名 杀死人物的名字

# 复活人物

复活某个人物

参数 描述
人物名 复活人物的名字

# 更换地图图片

更换地图的图片

参数 描述
图片文件名

# 修改景物图片

参数 描述
景物名
图片文件名 不填为移除当前景物的图片

# 激活障碍

参数 描述
障碍名

# 禁用障碍

参数 描述
障碍名
